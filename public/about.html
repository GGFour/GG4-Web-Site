ABOUT US

This page has been tailored by 5 engineering students from Oulu University of Applied Sciences, for the purpose of practicing our fundamentals in web-development. This project is a testament of our knowledge and skills we have acquired during our first year of studies. Before this project we’ve had courses on basics of JavaScript, HTML/CSS, Database programming and operating systems, softwares and programs for server management. We approached this project with the main goal of creating a user friendly, simple and flexible website, but later also doing something which challenges us as students. This site will in the future be a platform for hosting different games such as rogue-like dungeon crawlers and more. We are really excited to be working with this project and wish to receive any feedback. Please report also for any bugs you might encounter.
